## What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure and content of a webpage, using elements to define different parts of the page. 

### Key Concepts

- **Elements:** The building blocks of HTML, represented by tags (e.g., `<p>`, `<h1>`, `<a>`).
- **Tags:** Keywords enclosed in angle brackets that define the start and end of an element (e.g., `<p>This is a paragraph</p>`).
- **Attributes:** Provide additional information about an element, specified within the opening tag (e.g., `<a href="https://example.com">Link</a>`).
- **Structure:** HTML documents have a basic structure, including the `<html>`, `<head>`, and `<body>` elements.
